# list-home-skills

List installed home skill directories for local coding agents with readable, colorized output, or print a detailed skills table.

## Usage

```bash
scripts/list-home-skills/list-home-skills.sh
```

```bash
scripts/list-home-skills/list-home-skills-table.sh
```

The table script recursively inspects each skill directory and classifies provenance:
`local`, `private`, `public`, or `broken-link`.
